leap_year?(date)

Determines whether a date falls during a leap year. Leap years include all years divisible by 4, with the exception of years divisible by 100 but not divisible by 400 (got that?). That is, 2004 is a leap year, as is

  1. But while 2000 is a leap year (divisible by 400), 1900 is not

(divisible by 100 but not 400).

Source: show | on GitHub 

# File lib/rupee/day_count.rb, line 72
def leap_year?(date)  # :doc:
  year = date.year
  year % 4 == 0 && (year % 100 != 0 || year % 400 == 0)
end
